글에 붙인 키워드를 모아보는 공간입니다.

태그는 각 포스트의 front matter에서 지정합니다.

예를 들어 포스트 상단에 `tags: [jekyll, github-pages, test]`처럼 적으면 이 페이지에 태그 목록이 자동으로 나타납니다.

태그는 글의 세부 주제를 나타낼 때 좋습니다.
예를 들어 `jekyll`, `javascript`, `troubleshooting`, `daily-log`처럼 검색하기 쉬운 단어를 쓰면 됩니다.
