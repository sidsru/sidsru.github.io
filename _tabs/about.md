# About

안녕하세요. 이곳은 Sidsru 블로그의 소개 페이지입니다.

이 페이지에는 블로그 운영자 소개, 블로그에서 다룰 주제, 연락 가능한 링크를 자유롭게 적으면 됩니다.

## 다루고 싶은 주제

- 개발 공부 기록
- GitHub Pages와 Jekyll 운영 메모
- 프로젝트 진행 기록
- 문제 해결 과정 정리

## 블로그 운영 메모

이 파일은 `_tabs/about.md`입니다.  
본문을 수정하면 About 탭에 바로 반영됩니다.

예를 들어 자기소개를 바꾸고 싶다면 이 문단을 지우고 원하는 Markdown 문법으로 작성하면 됩니다.
